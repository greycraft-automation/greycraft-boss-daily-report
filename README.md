# Greycraft 老闆每日營收 LINE 報表（MVP）
### 自動整理每日營收，固定時間傳訊息給老闆。  

這是一個用 Python 寫的小工具，示範如何：

1. 從 `sales_sample.csv` 讀取每日銷售資料
2. 計算「今天」的營收總額與筆數
3. 輸出一段給老闆看的中文營收摘要訊息

目前版本：已可讀取示範銷售資料，並自動產生『今日營收快報』文字，可用於 Demo 給老闆看。  
未來會再加入：LINE 通知、自動排程等功能。

各位老闆您好，我是 Greycraft Automation（灰匠自動化）的 Jason。
我們不是賣 POS 機的，我主要做一件小事：  
**把您每天的營收，自動整理成一份一眼看得懂的「當日收款快報」，固定時間送到您的 LINE 或信箱。**

不管現在是用 POS、Excel 還是手寫帳，
很多店家每天都要：
- 從機器或平台抄數字
- 打開 Excel 算今天收多少
- 數字對不起來又重算一次  

我們做的這套小工具，是利用您「原本就有的資料」，
每天自動幫您整理三件事：  
1. 今天總共收多少？  
2. 各通路 / 各櫃檯 / 各店的表現？
3. 有沒有哪裡異常（例如：某一個通路特別低、退款特別多）？

老闆只要在固定時間打開 LINE / 信箱，
就看到一句話重點 + 一張簡單表格：

> Excel 函數是幫「填報的人」算數字，
這套工具是幫「您」把每天的數字變成一句話的營運結論，
兩個是互補，而不是互相取代。

我們目前已經有第一個導入的案例，可以用 Demo 給您看實際畫面。
如果方向對，我們會用**一次性的試點專案**來做：
- 依照您實際的帳務方式調整工具
- 上線前一起驗證數字
- 做好使用教學
- 上線後 1 個月追蹤與微調

試點專案通常在 **NT$ 15,000～20,000** 左右，做完是給您自己用的工具，
後面如果不用續約、不想加功能，也不會再額外收費。

想先請教您：
**現在旅館 / 餐廳 / 店裡，每天營收是怎麼看的？
大概會花誰的時間、花多久？**  
如果您願意分享一下，我可以先幫您判斷：  
這樣的自動化對您來說划不划算，不適合我也會直接說實話。
